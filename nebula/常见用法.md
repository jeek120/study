# 点
```
(a)
```

# 多点

```
# a到b的边
(a)-[]->(b)
# 任意数量的点和边
(a)-[]->(b)<-[]-(c)
```

# tag
```
(a:User)-[]->(b)
(a:User:Admin)-[]->(b)
```

# 属性
```
(a{name:'Andres', sport:'Brazilian Ju-Jitsu'})
(a)-[{blocked: false}] -> (b)
```

# 边
```
(a)-[]-(b)
(a)-[r]-(b)
(a)-[r:Edge_Type]-(b)
(a)-[r:TYPE1|TYPE2]-(b)
(a)-[:TYPE]-(b)
```
